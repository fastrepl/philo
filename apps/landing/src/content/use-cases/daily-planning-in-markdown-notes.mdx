Philo stores daily notes as markdown files on disk, but edits them in memory as TipTap JSON.

## Where this workflow fits

Philo is designed for people who already plan in markdown. The product direction is about keeping the loop between writing, deciding, and running as short as possible.

## Capability checklist

- Philo stores daily notes as markdown files on disk, but edits them in memory as TipTap JSON.
- Keep daily planning in one timeline.
- daily notes should help you run what you thought, not just remember it
- Philo is built to make daily notes lighter, not turn them into another subscription silo.
- Philo is the IDE for your daily notes.
